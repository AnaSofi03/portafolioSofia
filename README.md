## ⚙️ Instalación

### Clonar repositorio

```bash
git clone https://github.com/AnaSofi03/portafolioSofia.git
```

### Ingresar al proyecto

```bash
cd portafolioSofia
```

### Instalar dependencias

```bash
npm install
```

### Ejecutar proyecto

```bash
npm run dev
```
